# House Price Prediction

A Machine Learning project that predicts house prices using Linear Regression.

## Project Structure

```
House-Price-Prediction
│
├── data
├── notebook
├── model
├── images
├── src
├── README.md
├── requirements.txt
└── .gitignore
```

## Features

- Data Cleaning
- Exploratory Data Analysis
- Feature Selection
- Linear Regression
- RMSE Evaluation
- R² Score
- Model Saving
- Price Prediction

## Libraries

- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Run

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
